# Security Policy

## Supported Versions

We release patches for security vulnerabilities. Which versions receive security updates depends on the CVSS v3.0 Rating:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by emailing the maintainers directly or opening a security advisory on GitHub.

Please include the following information:
- Type of vulnerability
- Full paths of source file(s) related to the vulnerability
- The location of the affected code
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the vulnerability

We will respond within 48 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity but historically within a few days.
